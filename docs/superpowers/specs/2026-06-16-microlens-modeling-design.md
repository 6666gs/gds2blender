# 微透镜建模设计 (2026-06-16)

## 目标
用「修改脚本 → 后台跑 Blender 渲图 → AI 读图评估 → 调参重渲」的迭代闭环，
建出一个微透镜 3D 模型，对照用户提供的顶视图照片收敛形状。

## 几何意图（已与用户确认）
- **形状**：一个长方块，左右两个面被磨成**外凸球冠**；上/下/前/后保持平面。
  典型双凸折射微透镜。顶视图 = 矩形 + 左右外凸弧 + 上下直边（与照片一致）。
- **左右曲率不同**：左面凸得多（曲率大、半径小），右面较平缓（半径大），
  但**右面仍是球面，没有平面**。
- **尺寸**：按照片比例取默认值，µm 级（与项目 GDS 单位一致），渲图后微调。

## 构造方法：布尔交集（全参数化）
目标形状 = **长方块 ∩ 左球 ∩ 右球**
- 左球：球心在方块**右侧**远处、半径 `R_LEFT`，其左表面在方块左端形成朝 -x 外凸球冠。
- 右球：球心在方块**左侧**远处、半径 `R_RIGHT`，形成朝 +x 外凸球冠。
- 半径越小 → 凸得越狠。约束 `R_LEFT < R_RIGHT`。
- 上/下/前/后由原方块平面保留。

选布尔交集的理由：天然得到「方块+两球冠」精确形状；半径直接对应曲率，
调参直观；改一个数即可让某面更凸/更平，契合渲图迭代。
（备选：手写网格顶点 / 单球变形 —— 要么不稳，要么做不出矩形俯视轮廓，淘汰。）

## 默认参数（脚本顶部可调常量，单位 µm）
| 参数 | 默认 | 含义 |
|---|---|---|
| `BLOCK_X` (光轴/厚度) | 80 | 左右方向 |
| `BLOCK_Y` (深度) | 100 | 照片上下方向 |
| `BLOCK_Z` (高) | 50 | 竖直 |
| `R_LEFT` | 70 | 左面曲率半径（小→凸） |
| `R_RIGHT` | 160 | 右面曲率半径（大→平缓） |
| 球网格细分 | ≥128 段 | 保证球冠光滑 |

## 材质与渲染
- 透镜本体：玻璃 / 透明折射材质（复用现有脚本的 `use_screen_refraction` 手法）。
- 引擎：复用现成 Cycles 管线（可调采样 / 分辨率）。迭代用低采样草稿快渲。
- 背景：起步用中性 / 深色背景突出形状；是否复刻照片的橄榄色凹槽 + 蓝色高光，
  待形状收敛后再定。

## 迭代闭环
```
改 microlens.py → blender.exe --background --python microlens.py 渲草稿 PNG
   → AI 读 PNG，对照用户照片打分 → 调参数(半径/尺寸/材质) → 重渲
```
先收敛形状（草稿、低采样、快），满意后再上高采样精渲。

## 验收
- 顶视图轮廓与用户照片一致：左大弧 + 右缓弧 + 上下直边。
- 左右两面均为光滑球面，左凸右缓，无平面。
- 玻璃质感、深色背景出图清晰可辨形状。
