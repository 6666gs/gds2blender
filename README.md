# gds2blender

将芯片版图（GDS 格式）导入 Blender 并生成可渲染的 3D 结构。

## 脚本说明

本仓库包含四个独立的 Blender Python 脚本，按以下顺序执行：

### 1. `挂载gds导入.py` — GDS 核心导入器

在 Blender 侧边栏（N 键面板 → GDS Importer）提供完整的 UI 操作界面。

**功能：**
- 读取 GDS 文件，自动解析所有层（Layer/Datatype）
- 按层配置挤出方向：
  - **向上（生长/沉积）**：正向挤出，模拟薄膜沉积
  - **向下（刻蚀/挖槽）**：通过 2D 掩膜反转计算，精确模拟干法刻蚀
- 自动生成基底（Substrate），支持对 GDS 版图边界四个方向**独立**设置外扩增量（X-、X+、Y-、Y+）
- 每层可单独设置起始高度、厚度、颜色、启用状态

**依赖：** `gdspy`（需在 Blender 内置 Python 环境中安装）

```bash
# 在 Blender 的 Python 环境中安装
/path/to/blender/python/bin/pip install gdspy
```

**使用方式：**
1. 在 Blender 脚本编辑器中打开并运行此脚本（完成后面板常驻）
2. 在侧边栏选择 GDS 文件路径，点击「读取并分析 GDS 层」
3. 调整各层参数及基底设置
4. 点击「极速生成 3D 结构」

---

### 2. `光源与背景.py` — 工作室灯光与背景

一键配置适合芯片渲染的摄影棚灯光环境。

- 渲染引擎切换为 **Eevee**，开启 SSR、AO、Bloom
- 铺设超大深蓝灰背景底板（100,000 单位，覆盖任意尺寸芯片）
- 添加三盏太阳光：
  - **主光**（暖色，左前方）：主要照明
  - **轮廓光**（冷色，右后方）：勾勒波导与金属边缘反光
  - **顶光**（中性，正上方）：补充暗部细节
- 设置微弱全局环境光，消除纯黑死角

---

### 3. `添加摄影机.py` — 摄影机

添加并定位一个适合俯拍芯片的摄影机。

- 自动清理场景中已有摄影机
- 以 60° 俯角、45° 方位角放置，模拟产品展示视角
- 远裁剪面设为 100,000 单位，适配大尺寸版图

---

### 4. `转换为线条.py` — 线框渲染

将已生成的实体 GDS 模型转换为带光照的物理线框，用于线框风格渲染。

- 对 `GDS_Chip_Root` 下所有子网格添加 Wireframe 修改器
- 替换原有实体面，生成镂空线框
- 线框粗细默认 `0.5`（版图单位），可直接在脚本中修改

---

## 推荐工作流

```
1. 运行「挂载gds导入.py」→ 配置层参数 → 生成 3D 结构
2. 运行「光源与背景.py」     → 设置渲染环境
3. 运行「添加摄影机.py」     → 放置摄影机
4. （可选）运行「转换为线条.py」→ 切换为线框风格
5. 按 F12 渲染
```

## 环境要求

| 软件 | 版本要求 |
|------|----------|
| Blender | 3.6 及以上（推荐 4.x） |
| gdspy | 1.6 及以上 |

## 注意事项

- GDS 坐标单位默认为微米，可通过「缩放」参数调整
- 刻蚀层（向下）的 2D 掩膜计算基于 `gdspy.boolean`，复杂版图可能耗时数秒
- 基底外扩增量同时作用于刻蚀掩膜的边界，保持两者完全对齐
